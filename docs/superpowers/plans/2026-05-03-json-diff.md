# JSON Diff Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the JSON Diff placeholder with a fully functional semantic diff tool: two CodeMirror editors (left + right), a Rust-side diff engine that ignores object key order and compares arrays positionally, a virtualized result tree with status-driven coloring (Equal / Added / Removed / Modified / TypeChanged), default-collapse identical subtrees, "show only differences" toggle, search by path, and history save/load with `{ left, right }` payload.

**Architecture:** All diff computation happens in Rust (`json_diff` command building a `DiffTree`). Both inputs are first parsed via `serde_json::Value`, then walked recursively to produce a unified diff tree where each node carries a `DiffStatus`. The frontend stores `DiffTree` in a Zustand slice and renders a flattened-list virtualized tree with the same `react-window` pattern as the JSON Viewer. Closer rows are emitted by the same flatten convention introduced in the Viewer. The two editors reuse the existing `Editor.tsx` component from the JSON Viewer (extracted to a shared location).

**Tech Stack:** serde_json (existing) · `DiffTree` / `DiffStatus` domain types (NEW) · CodeMirror 6 (existing) · `react-window` `VariableSizeList` (existing) · Zustand (existing) · existing i18n + AppError + history infrastructure.

```rust
//! Semantic JSON diff: build a unified `DiffTree` describing how two
//! `serde_json::Value`s relate. Objects compared by key set (ignores
//! order); arrays compared positionally (no LCS in v1).

use crate::domain::path::{join_array_index, join_object_key};
use crate::error::AppError;
use serde::{Deserialize, Serialize};
use serde_json::Value;

#[derive(Serialize, Deserialize, Clone, Debug)]
pub struct DiffTree {
    pub root: DiffNode,
    pub stats: DiffStats,
}

#[derive(Serialize, Deserialize, Clone, Debug, Default)]
pub struct DiffStats {
    pub total_nodes: u32,
    pub differences: u32,
}

#[derive(Serialize, Deserialize, Clone, Debug)]
#[serde(tag = "kind", rename_all = "snake_case")]
pub enum DiffKey {
    Root,
    Object { name: String },
    Array { index: u32 },
}

/// What kind of value (or container) lives at this node, on either side.
/// `None` means the value is absent on that side.
#[derive(Serialize, Deserialize, Clone, Debug)]
#[serde(tag = "type", rename_all = "snake_case")]
pub enum DiffValue {
    Null,
    Bool { value: bool },
    Number { raw: String },
    String { value: String },
    /// Container present on at least one side. `children` carries the merged
    /// node list (including added/removed children).
    Object { key_count: u32 },
    Array { item_count: u32 },
}

#[derive(Serialize, Deserialize, Clone, Debug)]
#[serde(tag = "status", rename_all = "snake_case")]
pub enum DiffStatus {
    /// Both sides have identical primitives or congruent containers
    /// (recursion handles structural equality).
    Equal { value: DiffValue },
    /// Right has a value, left does not. `right` carries the new content.
    Added { right: DiffValue },
    /// Left has a value, right does not.
    Removed { left: DiffValue },
    /// Both primitives, same type, different value.
    Modified { left: DiffValue, right: DiffValue },
    /// Both sides present but incompatible types.
    TypeChanged { left: DiffValue, right: DiffValue },
}

#[derive(Serialize, Deserialize, Clone, Debug)]
pub struct DiffNode {
    pub id: u32,
    pub key: DiffKey,
    pub path: String,
    pub status: DiffStatus,
    pub children: Vec<DiffNode>,
    /// True if `self` differs OR any descendant differs. Used by the UI
    /// to default-collapse identical subtrees.
    pub has_difference: bool,
}

/// Public entry — build a `DiffTree` from two parsed JSON values.
pub fn build_diff(left: Value, right: Value) -> DiffTree {
    let mut ctx = BuildCtx::default();
    let root = ctx.diff_value(left, right, DiffKey::Root, String::new());
    DiffTree {
        stats: DiffStats {
            total_nodes: ctx.next_id,
            differences: ctx.differences,
        },
        root,
    }
}

/// Public entry to parse two raw strings and build the diff. Both sides
/// must be valid JSON; surfaces the first-failing parse error.
pub fn build_diff_from_strings(left: &str, right: &str) -> Result<DiffTree, AppError> {
    let left_v: Value = serde_json::from_str(left)?;
    let right_v: Value = serde_json::from_str(right)?;
    Ok(build_diff(left_v, right_v))
}

#[derive(Default)]
struct BuildCtx {
    next_id: u32,
    differences: u32,
}

impl BuildCtx {
    fn alloc_id(&mut self) -> u32 {
        let id = self.next_id;
        self.next_id += 1;
        id
    }

    fn diff_value(
        &mut self,
        _left: Value,
        _right: Value,
        _key: DiffKey,
        _path: String,
    ) -> DiffNode {
        unimplemented!("filled in by Task 3")
    }
}

// Helpers used by the algorithm (Task 3) — declared early so the next task
// has them ready.
pub(crate) fn describe(value: &Value) -> DiffValue {
    match value {
        Value::Null => DiffValue::Null,
        Value::Bool(b) => DiffValue::Bool { value: *b },
        Value::Number(n) => DiffValue::Number { raw: n.to_string() },
        Value::String(s) => DiffValue::String { value: s.clone() },
        Value::Object(map) => DiffValue::Object { key_count: map.len() as u32 },
        Value::Array(arr) => DiffValue::Array { item_count: arr.len() as u32 },
    }
}

pub(crate) fn type_label(value: &Value) -> &'static str {
    match value {
        Value::Null => "null",
        Value::Bool(_) => "bool",
        Value::Number(_) => "number",
        Value::String(_) => "string",
        Value::Object(_) => "object",
        Value::Array(_) => "array",
    }
}

#[allow(dead_code)] // used by Task 3
pub(crate) fn next_path_for_object(parent: &str, key: &str) -> String {
    join_object_key(parent, key)
}

#[allow(dead_code)] // used by Task 3
pub(crate) fn next_path_for_array(parent: &str, index: u32) -> String {
    join_array_index(parent, index)
}
```

```rust
// Object × Object: union of keys, recurse on each.
if let (Value::Object(left_map), Value::Object(right_map)) = (&left, &right) {
    let mut keys: Vec<String> = left_map.keys().cloned().collect();
    for k in right_map.keys() {
        if !left_map.contains_key(k) {
            keys.push(k.clone());
        }
    }
    keys.sort();

    let key_count_left = left_map.len() as u32;
    let key_count_right = right_map.len() as u32;
    let same_size = key_count_left == key_count_right;

    let mut children = Vec::with_capacity(keys.len());
    let mut any_diff = false;
    for k in &keys {
        let child_path = join_object_key(&path, k);
        let child = match (left_map.get(k).cloned(), right_map.get(k).cloned()) {
            (Some(lv), Some(rv)) => self.diff_value(
                lv,
                rv,
                DiffKey::Object { name: k.clone() },
                child_path,
            ),
            (None, Some(rv)) => self.make_added_node(rv, DiffKey::Object { name: k.clone() }, child_path),
            (Some(lv), None) => self.make_removed_node(lv, DiffKey::Object { name: k.clone() }, child_path),
            (None, None) => unreachable!("union of keys"),
        };
        if child.has_difference {
            any_diff = true;
        }
        children.push(child);
    }

    let status = if !any_diff && same_size {
        DiffStatus::Equal {
            value: DiffValue::Object {
                key_count: key_count_left,
            },
        }
    } else {
        // Container itself isn't "modified"; difference comes from children.
        DiffStatus::Equal {
            value: DiffValue::Object {
                key_count: key_count_left.max(key_count_right),
            },
        }
    };

    return DiffNode {
        id,
        key,
        path,
        status,
        children,
        has_difference: any_diff,
    };
}
```

Add the two `make_*_node` helpers as methods on `BuildCtx`:

```rust
fn make_added_node(&mut self, right: Value, key: DiffKey, path: String) -> DiffNode {
    let id = self.alloc_id();
    self.differences += 1;
    let described = describe(&right);
    let children = self.collect_descendant_for_added(right, &path);
    DiffNode {
        id,
        key,
        path,
        status: DiffStatus::Added { right: described },
        children,
        has_difference: true,
    }
}

fn make_removed_node(&mut self, left: Value, key: DiffKey, path: String) -> DiffNode {
    let id = self.alloc_id();
    self.differences += 1;
    let described = describe(&left);
    let children = self.collect_descendant_for_removed(left, &path);
    DiffNode {
        id,
        key,
        path,
        status: DiffStatus::Removed { left: described },
        children,
        has_difference: true,
    }
}

/// When a whole subtree was Added on the right, walk it and build child
/// `Added` nodes so the UI can render it expanded if the user wants.
fn collect_descendant_for_added(&mut self, value: Value, path: &str) -> Vec<DiffNode> {
    match value {
        Value::Object(map) => map
            .into_iter()
            .map(|(k, v)| {
                let child_path = join_object_key(path, &k);
                self.make_added_node(v, DiffKey::Object { name: k }, child_path)
            })
            .collect(),
        Value::Array(arr) => arr
            .into_iter()
            .enumerate()
            .map(|(i, v)| {
                let child_path = join_array_index(path, i as u32);
                self.make_added_node(v, DiffKey::Array { index: i as u32 }, child_path)
            })
            .collect(),
        _ => Vec::new(),
    }
}

fn collect_descendant_for_removed(&mut self, value: Value, path: &str) -> Vec<DiffNode> {
    match value {
        Value::Object(map) => map
            .into_iter()
            .map(|(k, v)| {
                let child_path = join_object_key(path, &k);
                self.make_removed_node(v, DiffKey::Object { name: k }, child_path)
            })
            .collect(),
        Value::Array(arr) => arr
            .into_iter()
            .enumerate()
            .map(|(i, v)| {
                let child_path = join_array_index(path, i as u32);
                self.make_removed_node(v, DiffKey::Array { index: i as u32 }, child_path)
            })
            .collect(),
        _ => Vec::new(),
    }
}
```

```ts
import { create } from "zustand";
import { diffApi } from "./api";
import { IpcError } from "../../lib/ipc";
import type { DiffTree } from "../../types/ipc";

export interface DiffDiagnostic {
  line: number;
  col: number;
  message: string;
}

interface JsonDiffState {
  left: string;
  right: string;
  tree: DiffTree | null;
  diffError: DiffDiagnostic | null;
  /** When true, hide subtrees with has_difference=false (default). */
  showOnlyDiffs: boolean;
  /** User-toggled collapse overrides on diff nodes. */
  collapseSet: Set<number>;
  searchQuery: string;
  loadedHistoryId: number | null;

  setLeft(text: string): void;
  setRight(text: string): void;
  setShowOnlyDiffs(v: boolean): void;
  setSearchQuery(q: string): void;
  toggleCollapse(id: number): void;
  swap(): void;
  clear(): void;
  compare(): Promise<void>;
  setLoadedHistoryId(id: number | null): void;
}

const initial = {
  left: "",
  right: "",
  tree: null as DiffTree | null,
  diffError: null as DiffDiagnostic | null,
  showOnlyDiffs: true,
  collapseSet: new Set<number>(),
  searchQuery: "",
  loadedHistoryId: null as number | null,
};

export const useJsonDiffStore = create<JsonDiffState>((set, get) => ({
  ...initial,

  setLeft(text) { set({ left: text }); },
  setRight(text) { set({ right: text }); },
  setShowOnlyDiffs(v) { set({ showOnlyDiffs: v }); },
  setSearchQuery(q) { set({ searchQuery: q }); },

  toggleCollapse(id) {
    const next = new Set(get().collapseSet);
    if (next.has(id)) next.delete(id);
    else next.add(id);
    set({ collapseSet: next });
  },

  swap() {
    const s = get();
    set({ left: s.right, right: s.left, tree: null, diffError: null });
  },

  clear() {
    set({ ...initial, collapseSet: new Set() });
  },

  async compare() {
    const { left, right } = get();
    if (left.trim().length === 0 && right.trim().length === 0) {
      return;
    }
    try {
      const tree = await diffApi.diff(left, right);
      set({
        tree,
        diffError: null,
        collapseSet: new Set(),
      });
    } catch (e) {
      if (e instanceof IpcError && e.app.code === "parse") {
        set({
          tree: null,
          diffError: {
            line: e.app.line,
            col: e.app.col,
            message: e.app.message,
          },
        });
      } else {
        const message = e instanceof Error ? e.message : String(e);
        set({ tree: null, diffError: { line: 0, col: 0, message } });
      }
    }
  },

  setLoadedHistoryId(id) { set({ loadedHistoryId: id }); },
}));
```

```ts
import type { DiffNode } from "../../types/ipc";

export interface DiffFlattenOptions {
  /** Node ids the user has explicitly collapsed. */
  collapseSet: Set<number>;
  /** When true, hide subtrees whose `has_difference === false`. */
  showOnlyDiffs: boolean;
  /** Substring filter against `node.path` (lowercase, empty disables). */
  searchQuery: string;
}

export interface VisibleDiffNode {
  node: DiffNode;
  depth: number;
  isCollapsed: boolean;
  isExpandable: boolean;
  /** Synthetic closer row for an expanded container. */
  closer?: "}" | "]";
}

export function flattenDiff(
  root: DiffNode,
  opts: DiffFlattenOptions,
): VisibleDiffNode[] {
  const out: VisibleDiffNode[] = [];
  const includeAll = opts.searchQuery.trim().length === 0;
  const q = opts.searchQuery.toLowerCase();
  const matchesSearch = (node: DiffNode): boolean =>
    includeAll || node.path.toLowerCase().includes(q);

  walk(root, 0, opts, matchesSearch, out);

  if (!includeAll) {
    // Search post-pass: keep only matches plus their ancestor chain (which
    // we approximate by keeping any node whose path is a prefix of a match
    // or a match is a prefix of its path).
    return restrictToSearch(out, q);
  }
  return out;
}

function restrictToSearch(
  rows: VisibleDiffNode[],
  q: string,
): VisibleDiffNode[] {
  const matchedPaths = new Set<string>();
  for (const r of rows) {
    if (r.closer) continue;
    if (r.node.path.toLowerCase().includes(q)) {
      matchedPaths.add(r.node.path);
    }
  }
  if (matchedPaths.size === 0) return [];

  const includedNodeIds = new Set<number>();
  const isAncestorOfMatch = (path: string) => {
    if (matchedPaths.has(path)) return true;
    for (const m of matchedPaths) {
      if (m === path) return true;
      if (path === "" && m !== "") return true;
      if (m.startsWith(path) && (m[path.length] === "." || m[path.length] === "[")) {
        return true;
      }
    }
    return false;
  };

  const filtered: VisibleDiffNode[] = [];
  for (const r of rows) {
    if (r.closer) continue;
    if (isAncestorOfMatch(r.node.path)) {
      filtered.push(r);
      includedNodeIds.add(r.node.id);
    }
  }
  // Re-add closers whose opener (same node.id) is included.
  for (const r of rows) {
    if (r.closer && includedNodeIds.has(r.node.id)) {
      filtered.push(r);
    }
  }
  // Sort to preserve original document order (closers naturally come after
  // openers since both share node.id; we tie-break by `closer` flag).
  return filtered.sort((a, b) => {
    if (a.node.id !== b.node.id) return a.node.id - b.node.id;
    return (a.closer ? 1 : 0) - (b.closer ? 1 : 0);
  });
}

function walk(
  node: DiffNode,
  depth: number,
  opts: DiffFlattenOptions,
  matches: (n: DiffNode) => boolean,
  out: VisibleDiffNode[],
): void {
  // showOnlyDiffs filter: skip whole subtree if neither self nor descendants differ.
  if (opts.showOnlyDiffs && !node.has_difference) {
    return;
  }
  // search prefilter: skip pure misses (only the search post-pass restores ancestors).
  // We still emit here because the post-pass depends on parent context.

  const isCollapsed = opts.collapseSet.has(node.id);
  const expandable = (node.children?.length ?? 0) > 0;

  out.push({
    node,
    depth,
    isCollapsed,
    isExpandable: expandable,
  });
  // suppress unused warning while keeping the API symmetric
  void matches;

  if (isCollapsed) return;
  if (!node.children || node.children.length === 0) return;

  for (const child of node.children) {
    walk(child, depth + 1, opts, matches, out);
  }

  // Emit closer row for the container.
  const isObject =
    node.status === "equal" && node.value?.type === "object" ||
    node.status === "added" && node.right?.type === "object" ||
    node.status === "removed" && node.left?.type === "object" ||
    (node.status === "modified" || node.status === "type_changed") && false;
  // Modified/TypeChanged primitives never get container closers; only true
  // containers get one. Determine via children presence + value-shape.
  const isArray =
    node.status === "equal" && node.value?.type === "array" ||
    node.status === "added" && node.right?.type === "array" ||
    node.status === "removed" && node.left?.type === "array";

  if (isObject) {
    out.push({
      node,
      depth,
      isCollapsed: false,
      isExpandable: false,
      closer: "}",
    });
  } else if (isArray) {
    out.push({
      node,
      depth,
      isCollapsed: false,
      isExpandable: false,
      closer: "]",
    });
  }
}
```

```tsx
function highlight(text: string, query: string): React.ReactNode {

function summarize(value: DiffValue | undefined): string {
  if (!value) return "";
```

```tsx
const ROW_HEIGHT = 28;

function useElementSize(): [
  React.RefObject<HTMLDivElement>,
  { width: number; height: number },
] {
  const ref = useRef<HTMLDivElement>(null);
  const [size, setSize] = useState({ width: 0, height: 0 });
  useEffect(() => {
    const el = ref.current;
    if (!el) return;
    const ro = new ResizeObserver((entries) => {
      for (const entry of entries) {
        const { width, height } = entry.contentRect;
        setSize({ width, height });
      }
    });
    ro.observe(el);
    return () => ro.disconnect();
  }, []);
  return [ref, size];
}

export function DiffPane() {

  const tree = useJsonDiffStore((s) => s.tree);

  const [hostRef, size] = useElementSize();

  const visible: VisibleDiffNode[] = useMemo(() => {
    if (!tree) return [];
    return flattenDiff(tree.root, {
      collapseSet,
      showOnlyDiffs,
      searchQuery,
    });

  const renderRow = ({ index, style }: ListChildComponentProps) => {
    const v = visible[index];

            style={{
              height: ROW_HEIGHT,
              paddingLeft: `${v.depth * 16 + 8 + 16}px`,
            }}
          >
            {v.closer}
          </div>
        </div>
      );
    }
    return (
      <div style={style}>
        <DiffNode
          visible={v}
          onToggleCollapse={(id) => {
            toggleCollapse(id);
            listRef.current?.resetAfterIndex(0);
          }}
          searchQuery={searchQuery}
        />
      </div>
    );
  };

  return (
    <div ref={hostRef} className="h-full w-full overflow-hidden relative">
      {!tree && (
        <div className="absolute inset-0 flex items-center justify-center text-[color:var(--text-muted)] text-sm pointer-events-none">
          {t("json_diff.tree_empty")}
        </div>
      )}

          ref={listRef}
          height={size.height}
          width={size.width}
          itemCount={visible.length}
          itemSize={() => ROW_HEIGHT}
          overscanCount={20}
        >
          {renderRow}
        </VariableSizeList>
      )}
```

```tsx
import { useState } from "react";
import { useTranslation } from "react-i18next";
import { SaveDialog, type SaveResult } from "../../history/SaveDialog";
import { useJsonDiffStore } from "./store";
import { useHistoryStore } from "../../history/store";
import { useToastStore } from "../../shell/toastStore";
import type { SaveRequest } from "../../types/ipc";

export function SaveDialogHost({
  open,
  onClose,
}: {
  open: boolean;
  onClose: () => void;
}) {
  const { t } = useTranslation();
  const left = useJsonDiffStore((s) => s.left);
  const right = useJsonDiffStore((s) => s.right);
  const loadedHistoryId = useJsonDiffStore((s) => s.loadedHistoryId);
  const setLoadedHistoryId = useJsonDiffStore((s) => s.setLoadedHistoryId);
  const save = useHistoryStore((s) => s.save);
  const push = useToastStore((s) => s.push);

  const [defaultTitle] = useState(() => {
    const head = (left + " vs " + right).slice(0, 50).replace(/\s+/g, " ").trim();
    return head || t("json_diff.save_default_title_left");
  });

  const handle = async (result: SaveResult) => {
    const req: SaveRequest =
      result.mode === "new"
        ? {
            mode: "new",
            title: result.title,
            content: { tool: "json_diff", left, right },
          }
        : {
            mode: "overwrite",
            id: result.id,
            title: result.title,
            content: { tool: "json_diff", left, right },
          };
    try {
      const item = await save(req);
      setLoadedHistoryId(item.id);
      onClose();
    } catch (e) {
      push("error", e instanceof Error ? e.message : String(e));
    }
  };

  return (
    <SaveDialog
      open={open}
      defaultTitle={defaultTitle}
      loadedId={loadedHistoryId}
      onSave={handle}
      onCancel={onClose}
    />
  );
}
```
